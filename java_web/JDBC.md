# JDBC API
DriverManager 类：管理各种不同的 JDBC 驱动。驱动是java程序和数据库的桥梁

Connection 接口：连接 MySQL 数据库

Statement 接口：执行 SQL 语句

ResultSet 接口：接收结果集
